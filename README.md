# MoxyClass
An abstraction for Lua classes

### Support:
- Fields
- Methods
- Inheritance
- Constructor
- Type checking
- IsA method (inheritance check)

### Future support:
- Static methods
- Static fields
- Private fields
- Private methods
- Public fields
- Public methods
- Getters and setters
- Events
- Enumerations
- Interfaces
- Abstract classes
- Abstract methods
- Operator overloading
- Type casting
- Type conversion
- Type checking
- Type inference
